## What is doubleespresso-films?

Doubleespresso-films is a modern web application built with Next.js 15 that provides a curated catalog of films with ratings, reviews, and integrated video playback. The platform allows users to browse films, view detailed information, and watch content directly in the browser.

## Who is it for?

This application is designed for:

- **Film enthusiasts** who want to discover and watch curated content
- **Content curators** managing film collections with ratings and metadata
- **Developers** looking to build or extend a modern Next.js film catalog platform

## Key features

### Film catalog
Browse a comprehensive collection of films with poster images, ratings, genres, and watch dates. The catalog supports search and multiple sorting options.

### Advanced video player
Custom-built video player with features including:
- Playback speed control (0.5x to 2x)
- Keyboard shortcuts (Space, Arrow keys)
- Fullscreen support
- Volume control
- Progress tracking
- Optional chat overlay for synchronized commentary

### Responsive design
Fully responsive interface that works seamlessly across desktop, tablet, and mobile devices with adaptive controls.

### Theme support
Built-in dark and light theme support with automatic detection of system preferences and persistent user selection.

### Content management integration
Integrates with Strapi CMS (admin.doublekava.watch) for dynamic content management, allowing easy updates to the film catalog without code changes.

### Analytics
Google Analytics integration for tracking user engagement and viewing patterns.

## Technology stack

- **Framework**: Next.js 15 with App Router
- **UI**: React 19, Tailwind CSS 4, Radix UI components
- **TypeScript**: Full type safety throughout the application
- **Fonts**: Geist Sans and Geist Mono from Vercel
- **Icons**: Heroicons and Lucide React
- **Analytics**: Google Analytics via react-gtag
